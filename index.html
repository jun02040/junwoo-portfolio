About Me

저는 한라대학교 컴퓨터공학과에서 다양한 전공 수업을 수강하며 프로그래밍과 웹 개발의 기초를 학습했습니다.

사용자에게 편리한 서비스를 제공하는 웹 개발에 관심이 있으며, 새로운 기술을 배우고 문제를 해결하는 과정을 즐깁니다.

졸업작품

코딩 타자 연습 웹사이트

코딩이 생소하거나 처음 접하는 사람들이 코드 입력을 통해 타자 연습과 프로그래밍 학습을 동시에 할 수 있도록 제작한 웹사이트입니다.

기술 스택

React Node.js Express MySQL

목적

일반 타자 연습이 아닌 코드 문장을 입력하면서 프로그래밍 문법과 타이핑에 자연스럽게 익숙해질 수 있도록 제작했습니다.

주요 기능

  • 언어 선택 (C / Python / Java)
  • 짧은 글 / 긴 글 / 미니게임 모드
  • 타자 속도 측정
  • 웹 기반 학습 환경 제공
  • 사용자 기록 저장 및 관리

개발 과정

React를 활용하여 사용자 인터페이스를 구현하고, Node.js와 Express를 이용하여 서버를 구축했습니다. 또한 MySQL 데이터베이스를 연동하여 사용자 기록과 정보를 관리할 수 있도록 개발했습니다.

배운 점

프론트엔드와 백엔드를 모두 개발하면서 웹 서비스의 전체적인 동작 구조를 이해할 수 있었습니다. 또한 데이터베이스 설계와 API 연동을 경험하며 문제 해결 능력과 개발 역량을 향상시킬 수 있었습니다.
