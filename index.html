About Me

저는 한라대학교 컴퓨터공학과에서 다양한 전공 수업을 들으며 프로그래밍과 웹 개발의 기초를 학습했습니다. 이 사이트는 대학 생활 동안 배운 내용과 졸업작품을 정리한 포트폴리오입니다.

인공지능

인공지능의 기본 개념과 문제 해결 방법을 학습했습니다.

웹프로그래밍

HTML, CSS, JavaScript를 이용한 웹사이트 제작

데이터베이스응용

데이터 저장 구조와 SQL 활용을 배웠습니다.

기타 전공

전공 수업을 통해 프로그래밍 능력을 향상했습니다.

졸업작품

코딩 타자 연습 웹사이트

코딩이 생소하거나 처음 접하는 사람들이 코드 입력을 통해 타자 연습을 할 수 있도록 만든 웹사이트입니다.

목적

일반 타자 연습이 아닌 코드 문장을 입력하면서 프로그래밍에 익숙해지도록 돕는 것이 목표입니다.

주요 기능

  • 언어 선택 (C / Python / Java)
  • 짧은 글 / 긴 글 / 미니게임
  • 타자 연습 기능
  • 웹 기반 실행

사용 기술

HTML / CSS / JavaScript

배운 점

웹페이지 구조 설계와 사용자 화면 구성을 직접 만들면서 웹 개발의 기본을 경험할 수 있었습니다.
