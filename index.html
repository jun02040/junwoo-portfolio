코딩 타자 연습 웹사이트

코딩 문장을 입력하며 타자 속도와 정확도를 향상시키는 웹사이트입니다.

React Node.js Express MySQL

목적

프로그래밍 문장 입력을 통해 코드 친숙도 및 타자 실력 향상

주요 기능

  • 언어 선택 (C / Python / Java)
  • 타자 속도 측정
  • 문장 랜덤 출력
  • 사용자 기록 저장

개발 과정

React로 프론트엔드를 구성하고 Node.js + Express로 서버를 구현했습니다. MySQL을 이용해 사용자 기록을 저장하도록 설계했습니다.

배운 점

프론트엔드와 백엔드를 연결하는 구조를 이해했고, API 통신과 데이터베이스 설계 경험을 쌓을 수 있었습니다.
